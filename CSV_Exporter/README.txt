Running GamedayExport.py

Use this utility to export game day data to a CSV file. 

1. Before you run GamedayExport.py, please run the following command to install dependencies. 

    pip install -r requirements.txt

2. To run the script, open a terminal (Mac/Linux) or command prompt window (Windows) and type the following command.

    python3 GamedayExport.py

3. The script will ask for a date. Please enter the date in the YYYY-MM-DD format. 

4. The CSV file will be saved in the same directory as the script. Your requested date will be the prefix to a _Gameday_Data.csv file. 

    Example: 2016-08-04_Gameday_Data.csv

NOTE: If at any time you want to cancel the program's execution, press Ctrl + C on both Windows and Mac/Linux systems.
